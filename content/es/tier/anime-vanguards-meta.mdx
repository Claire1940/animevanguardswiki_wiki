### Sistema de piedad de rasgos (2026)
El sistema de piedad asegura que los jugadores eventualmente obtengan los rasgos de alto nivel que necesitan. Ten en cuenta que la piedad se rastrea por separado para cada unidad y no se reinicia al evolucionar o transferir estadísticas.

<Steps>
  <Step num="1" title="Piedad de Solar">
    Requiere **300 Rerolls**. Este es el rasgo mítico de nivel inicial para un daño de AoE consistente.
  </Step>
  <Step num="2" title="Piedad de Deadeye">
    Requiere **400 Rerolls**. Esencial para unidades que dependen de golpes críticos para superar jefes de alta defensa.
  </Step>
  <Step num="3" title="Piedad de Ethereal">
    Requiere **858 Rerolls**. Este es el estándar para unidades de daño competitivas de endgame.
  </Step>
  <Step num="4" title="Piedad de Monarch">
    Requiere **1500 Rerolls**. El objetivo final para unidades como Rogita (Super 4) o Alocard.
  </Step>
</Steps>

</Section>

<Section icon={Shield} title="Progresión del meta: Evolución y Ascensión">

La evolución es el método principal para aumentar el techo de poder de una unidad. En el **meta de anime vanguards** de 2026, simplemente tener una unidad Mítica ya no es suficiente; debes evolucionarlas a sus formas Secret o Vanguard para desbloquear todo su potencial.

| Tipo de mejora | Requisito de rareza | Beneficio principal | Necesidad en el meta |
| :--- | :--- | :--- | :--- |
| **Evolución** | Cualquiera | Nuevas habilidades/pasivas | Obligatorio |
| **Ascensión** | Solo Mítica | Bonificaciones de estadísticas fijas | Opcional/Alto nivel |
| **Memoria** | Todas | Mejoras/enlaces únicos | Alto |

<Steps>
  <Step num="1" title="Selecciona objetivos del meta">
    Elige unidades que definan el meta actual de 13.5, como el núcleo de Cazadores de Demonios o las nuevas unidades de Verano.
  </Step>
  <Step num="2" title="Cultiva materiales de evolución">
    Recoge piedras de esencia y oro. Usa las **Etapas de Leyenda** y **Raids** para encontrar objetos específicos de unidad.
  </Step>
  <Step num="3" title="Completa misiones de evolución">
    Unidades como **Elastic Captain** requieren misiones específicas de varios pasos. Termínalas antes de intentar el área de mejora.
  </Step>
  <Step num="4" title="Ascender unidades Míticas">
    Usa al NPC Kokashi para añadir hasta un 10% de daño extra a través de tres niveles de ascensión. **Nota**: La ascensión se elimina si una unidad Mítica evoluciona a una rareza superior.
  </Step>
</Steps>
