### Sistema de Piedade de Atributos (2026)
O sistema de piedade garante que os jogadores eventualmente consigam os atributos de alto nível de que precisam. Observe que a piedade é rastreada separadamente para cada unidade e não é redefinida após a evolução ou transferências de status.

<Steps>
  <Step num="1" title="Piedade Solar">
    Requer **300 Rerolls**. Este é o atributo mítico de nível de entrada para dano em área consistente.
  </Step>
  <Step num="2" title="Piedade Deadeye">
    Requer **400 Rerolls**. Essencial para unidades que dependem de acertos críticos para contornar chefes com alta defesa.
  </Step>
  <Step num="3" title="Piedade Ethereal">
    Requer **858 Rerolls**. Este é o padrão para carries competitivos de endgame.
  </Step>
  <Step num="4" title="Piedade Monarch">
    Requer **1500 Rerolls**. O objetivo final para unidades como Rogita (Super 4) ou Alocard.
  </Step>
</Steps>

<Section icon={Shield} title="Progressão de Meta: Evolução e Ascensão">

A evolução é o método principal de aumentar o teto de poder de uma unidade. No **meta de anime vanguards** de 2026, simplesmente ter uma unidade Mythic não é mais suficiente; você deve evoluí-las para suas formas Secret ou Vanguard para desbloquear seu potencial total.

| Tipo de Melhoria | Requisito de Raridade | Benefício Principal | Necessidade no Meta |
| :--- | :--- | :--- | :--- |
| **Evolução** | Qualquer | Novas habilidades/passivas | Obrigatório |
| **Ascensão** | Apenas Mythic | Bônus de status fixos | Opcional/High-end |
| **Memoria** | Todos | Buffs/links únicos | Alto |

<Steps>
  <Step num="1" title="Selecione Alvos Meta">
    Escolha unidades que definem o meta 13.5 atual, como o core de Demon Hunter ou as novas unidades de Verão.
  </Step>
  <Step num="2" title="Farme Materiais de Evolução">
    Colete pedras de essência e ouro. Use os **Legend Stages** e **Raids** para encontrar itens específicos de unidades.
  </Step>
  <Step num="3" title="Complete Missões de Evolução">
    Unidades como **Elastic Captain** exigem missões específicas de várias etapas. Conclua-as antes de tentar a área de melhoria.
  </Step>
  <Step num="4" title="Ascenda Unidades Mythic">
    Use o NPC Kokashi para adicionar até 10% de dano extra através de três níveis de ascensão. **Nota**: A ascensão é removida se uma unidade Mythic evoluir para uma raridade superior.
  </Step>
</Steps>

A atualização 13.5 simplificou o sistema de Memoria. Todas as Memorias agora são evoluídas por padrão, e você pode usar a **Reciclagem de Memoria** para transformar fragmentos indesejados em **Memoria Scrap** para melhores upgrades.
</Callout>
